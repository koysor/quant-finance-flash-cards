## Definition

The **Gap Statistic** (Tibshirani, Walther & Hastie, 2001) is a principled method for selecting the optimal number of clusters $K$ by comparing the observed within-cluster dispersion to what would be expected under a null reference distribution of data with no cluster structure. It selects the smallest $K$ for which the within-cluster dispersion is substantially lower than the null expectation.

## Key Formula

For a given $K$, compute the log inertia $W_K = \log J(K)$ and compare to the expected value under $B$ random reference datasets:

$$\text{Gap}(K) = \mathbb{E}^*[\log W_K] - \log W_K$$

where $\mathbb{E}^*[\log W_K] = \frac{1}{B}\sum_{b=1}^{B} \log W_K^{(b)}$ over $B$ uniformly distributed reference datasets.

**Selection rule** — choose the smallest $K$ satisfying:

$$\text{Gap}(K) \geq \text{Gap}(K+1) - s_{K+1}$$

where $s_{K+1}$ is the standard deviation of the reference distribution at $K+1$. This one-standard-error rule guards against selecting spurious extra clusters.

## Example

Detecting market regimes from 10 years of macro features. Elbow plot is ambiguous (smooth decline in inertia). Gap statistic: $B=50$ reference datasets generated by sampling uniformly within the data's bounding box. Gap(1)=0.4, Gap(2)=0.8, Gap(3)=1.1, Gap(4)=1.05 — Gap drops after $K=3$. One-SE rule confirms $K=3$ (bull, bear, crisis), overriding the ambiguous elbow.

## Remember

The gap statistic is the rigorous alternative to the elbow method when financial data shows no clear elbow — which is common because financial return distributions blend smoothly between regimes rather than clustering tightly. By comparing observed dispersion to a null reference, the gap statistic answers the question "does this data actually have cluster structure?" rather than just "where does the inertia curve bend?" The one-standard-error rule prevents over-specifying the number of market regimes, which leads to unstable regime labels that flip frequently in live trading.
